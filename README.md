# BOT DE DETECCIÓN DE REENVÍOS

**CONFIGURACIÓN RÁPIDA:**
1. Clona el repo
2. En `config.json` pon tu número en "OWNER"
3. Ejecuta `npm install`
4. Inicia con `npm start`
5. Para parar: `Ctrl + C`

**COMANDOS:**
- `.onov` → Activa detector
- `.offov` → Desactiva detector

**¡SOLO PON TU NÚMERO Y DISFRUTA!** :v
